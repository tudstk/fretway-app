# Guitar Progress App

Flutter mobile app for tracking guitar practice progress, goals, and achievements.

## Modules

### Auth
- Login screen with email and password
- Registration screen
- Forgot password screen
- Landing screen

### Goals
- Goals list with progress indicators
- Goal detail view with milestone timeline
- Create and edit goals
- Add milestones to goals
- Track milestone completion

### Training
- Practice sets management
- Exercises library with predefined and custom exercises
- Create custom practice sets
- Filter exercises by difficulty

### Practice
- Start practice sessions from practice sets
- Active session screen with timer
- Track practice time and exercises completed
- Session completion screen with notes

### Metrics
- User level and XP tracking
- Streaks and badges display
- Goals progress overview
- Practice session history
- Achievements screen with badge gallery

### Profile
- User profile display
- Settings screen
- Logout functionality
